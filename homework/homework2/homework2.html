What is a dataset?

A dataset is a structured collection of data gathered and organized for analysis. It can be a table with rows (observations) and columns (variables), a time series, or even a collection of images or texts. Each row typically represents an observation (e.g., an event), while each column represents a variable or feature of that observation.

What is a distribution?

In statistics, a distribution describes how data are spread across the possible values—how often different values or ranges occur. Understanding the distribution lets us see where the data cluster, where they are sparse, and the overall shape of the dataset.

Example of a real dataset and distributions

Let’s imagine working in a security operations center that monitors access logs to a corporate portal. Every time a user interacts with the system, an event is recorded describing what happened. From this collection of events, we can build a dataset, a structured set of data designed for statistical analysis. Suppose the dataset contains three main columns: a user identifier (user_id), a precise timestamp of the event (timestamp), and the type of action performed (action).
